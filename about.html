Why?

Snowball was one of my favourite games from my childhood. Played as the text only version on a BBC Model B in the early eighties it's one game that left a lasting impression on me.

How?

The Level9 interpreter was originally written in C by Glen Summers, and later extended by David Kinder, along with Alan Staniforth, Simon Baldwin, Dieter Baron and Andreas Scherrer. This C code has been (crudely) ported to Java and then used as the game engine for the Android application.

What next?

No plans as it was Snowball that I wanted to play, but Return to Eden and Worm in Paradise would very easy to do, as would any of the A code games as the interpreter should support them all.
